# Probable Pancake

Tools for hacking. Build with python, kivy, sqlite3

## Feature

- Request API
- Facebook Bruteforce
- Zip Bruteforce
- FTP Bruteforce
- DDOS Server
- Check IP Server
- Check Open Port Server
- Auto Save To Database

## Installation

```bash
git clone https://github.com/ferdiansyah0611/probable-pancake --depth 1
cd probable-pancake
pip install -r pip.txt
```

## For GUI

```bash
cd probable-pancake
pip install kivy
py gui.py
```

## Usage

```bash
python app.py
```
